O formulário do Marketplace só aceita o modo **Visual**. Cole o bloco abaixo (texto puro). Depois selecione os títulos e use **Bold** / lista com bullets na toolbar, se quiser.

```
Builder Languages for Breakdance adds professional language packs for the Breakdance Builder interface, Breakdance admin screens, and first-party Breakdance Elements — without editing Breakdance core files.

American English (en_US) stays native to Breakdance. This plugin ships English International (en_GB) plus 16 additional product locales.

Included languages
Portuguese (Brazil & Portugal)
Italian, French, German
Spanish (Spain & Latin America)
Dutch, Polish, Russian
Arabic, Hebrew, Hindi
Japanese, Korean, Simplified Chinese
English International (en_GB)

Features
Builder UI translations
Admin / PHP translations
Breakdance Elements translations
Form Builder related strings
RTL support (Arabic / Hebrew)
Per-user language via Breakdance → Languages

Important scope
Does not translate front-end website content (posts, pages, WooCommerce products).
Use WPML, Polylang, TranslatePress, or Weglot for content translation.
Independent companion by UX Widget — not affiliated with, endorsed by, or sponsored by Breakdance / Soflyy.

Purchase, licensing, and updates are handled via Freemius.
Product page: https://uxwidget.com/builder-languages-breakdance/
```
