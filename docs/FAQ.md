# Breakdance Languages - FAQ

## Does this plugin require Breakdance Builder?

Yes. Breakdance Languages is built specifically for Breakdance Builder and its first-party elements.

## Is Breakdance Languages made by Breakdance?

No. Breakdance Languages is an independent plugin by UX Widget. It is not affiliated with, endorsed by, or maintained by the Breakdance team.

## Does it translate the Breakdance Builder interface?

Yes. The plugin loads JavaScript translation files for the Breakdance Builder interface and gettext files for PHP/admin screens.

## Does it translate WooCommerce?

WooCommerce strings are usually handled by WooCommerce language packs because they use the `woocommerce` text domain. Breakdance Languages focuses on Breakdance and Breakdance Elements.

## Why do some texts still appear in English?

Some texts can be hardcoded in compiled JavaScript, generated by third-party plugins, stored in your content, or loaded from another text domain. Those strings may require a targeted compatibility update.

## My profile is English but I chose Portuguese in Breakdance Languages. Why was the builder still partly in English?

Older versions required manual alignment. Current versions **sync the WordPress profile locale** when you pick an explicit builder language (not Auto). Choose `Português (Brasil)` in `Breakdance > Languages`, reload the builder, and both profile and builder should match.

If you use **Use WordPress profile language**, the builder follows the profile — change the profile under `Users > Profile > Language`.

## Can each admin use a different language?

Yes. On Pro, each user can choose a builder language in `Breakdance > Settings > Languages`. On Free, the plugin follows the WordPress profile language when that locale is included in the free plan.

## Will updates overwrite my corrections?

If you directly edit plugin files, updates can overwrite those edits. Keep a record of manual corrections and apply them to the next release package, or request inclusion in the official language pack.

## Does this plugin use AI to translate live text?

The current plugin loads prebuilt translation files. A future version may include AI-assisted translation review or missing-string suggestions.

## Can I use this on client websites?

Yes, according to the license plan you purchase. Agency plans are intended for client work and multiple sites.
